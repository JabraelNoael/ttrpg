Biome taxonomy — draft for redline

31 biomes across 8 families. The badge on each swatch is its generation weight (higher = more common). Each biome carries temperature + moisture tags — these drive weather (e.g. snow favors cold) so weather.json never needs a per-biome table. Affinity = the biomes it likes to border, which grows continuity during /map generate. Edit biomes_draft.json directly: rename, cut, recolor, retag, reweight.
